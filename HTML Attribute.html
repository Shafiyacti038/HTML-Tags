HTML Attributes

  • All HTMl elements can have attributes. Attributes provide additional information about elements.
  • Attributes are always specified in the start tag. Attributes usually come in name/value pairs like:name="value"

The href Attribute

The <a> tag defines a hyperlink. The href attribute specifies the URL of the page.

Syntax:

<a href="https://www.w3schools.com"> Visit w3schools</a>

Visit w3schools

The src Attribute

The <img> tag is used to embed an image in an HTML page. The src attribute specifies the path to the image to the image to be displayed.

The <img> tag should also contain the width ,height and alt attributes,which specify the width,height and alternate text of an image(in pixels).

Syntax

<img src="images.jpg" width="500" height="600" alt ="HTML">

The Lang Attribute

The lang attribute is used to declare the language of the webpage.

Syntax:

<html lang="en-US">
